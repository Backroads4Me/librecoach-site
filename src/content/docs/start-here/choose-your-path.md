Libre Coach offers three installation paths. Choose the one that fits your comfort level, timeline, and budget.

## Quick Comparison

| Path | Cost | Time | Difficulty | Best For |
|------|------|------|------------|----------|
| **DIY Build** | $150-200 | 2-3 hours | Medium | Hands-on learners |
| **Manual Install** | $150-200+ | 4-6 hours | Hard | Complete control & customization |
| **Pre-Assembled Kit** | $300-350 | 30 min | Easy | Plug-and-play convenience |

---

## Path 1: DIY Build (Recommended)

### What You Do
1. Order parts from Amazon/AliExpress (~$150-200)
2. Assemble hardware following our step-by-step guide
3. Flash pre-configured image to NVME drive
4. Connect to your RV and configure WiFi

### Pros
- **Fastest path to get started** (parts arrive in days)
- Pre-configured image = no manual software setup
- Learn how it works while building
- Best value for money

### Cons
- Requires basic assembly (screwdrivers, following instructions)
- Need to flash image to NVME drive

### Best For
- Users who enjoy hands-on projects
- Anyone who wants to understand their system
- Those who want to get started quickly without waiting for a kit

## Path 2: Manual Install (Advanced)

### What You Do
1. Order or use existing Raspberry Pi hardware
2. Assemble hardware
3. Manually install Home Assistant OS
4. Install and configure each add-on individually
5. Import Libre Coach flows into Node-RED
6. Connect to your RV

### Pros
- **Complete control** over every component
- Use existing hardware if you have it
- Understand the entire software stack
- Can customize everything during installation

### Cons
- **Time-intensive** (4-6 hours for first-time install)
- Requires technical knowledge (Linux, Home Assistant, MQTT)
- More opportunities for configuration errors
- Must troubleshoot issues independently

### Best For
- Experienced Home Assistant users
- Those installing on existing hardware
- Developers who want to modify the system
- Users who want to understand every component

## Still Not Sure?

### Choose DIY Build if:
- ✅ You want to get started this week
- ✅ You're comfortable following assembly guides
- ✅ You want the best value
- ✅ You don't mind some light hardware assembly

### Choose Manual Install if:
- ✅ You're already familiar with Home Assistant
- ✅ You want to learn the entire system deeply
- ✅ You have existing hardware to use
- ✅ You enjoy technical challenges

### Choose Pre-Assembled Kit if:
- ✅ You want zero assembly required
- ✅ You prefer tested, ready-to-use hardware
- ✅ You can wait for batch availability
- ✅ Budget isn't your primary concern

## What Happens After Installation?

Regardless of which path you choose, the end result is the same: a fully functional Libre Coach system. After installation, you'll:

1. **Connect to your RV's CAN bus** ([CAN Connection Guide](/installation/can-connection/))
2. **Identify your devices** by toggling switches ([Identifying Switches](/customization/identify-switches/))
3. **Create custom dashboards** tailored to your RV ([Dashboards Guide](/customization/dashboards/))
4. **Set up automations** to make your RV smarter ([Automation Examples](/customization/automation/))
