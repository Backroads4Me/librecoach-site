The LibreCoach community hangs out in two places:

- <a href="https://www.facebook.com/groups/librecoach/" target="_blank" rel="noopener noreferrer">**Facebook Group**</a> — General discussion, show-and-tell, and support.
- <a href="https://discord.gg/VZCAESHn2h" target="_blank" rel="noopener noreferrer">**Discord Server**</a> — Real-time chat and quick questions.

## Asking for Help

Search first — many problems have already been solved. When posting a question, include:

1. **RV** — Year, make, model (e.g., _2022 Tiffin Allegro Bus 45OPP_)
2. **Hardware** — Pi model, CAN HAT type (e.g., _Pi 4, Waveshare RS485/CAN HAT_)
3. **Software** — Home Assistant and LibreCoach versions
4. **The problem** — What you expected vs. what happened
5. **Logs** — Relevant app logs

## Bug Reports & Contributions

Open an issue or pull request on <a href="https://github.com/backroads4me" target="_blank" rel="noopener noreferrer">**GitHub**</a>.

## Private Inquiries

For security disclosures or administrative matters, email **hello@librecoach.com**.

## Community Guidelines

This community is here to build cool stuff for RVs. Be respectful, be constructive, and assume positive intent. Help others when you can — someone helped you get started too.
