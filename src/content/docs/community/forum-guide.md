## How to Ask Good Questions

**Please SEARCH first!**
Before posting, use the search bar. There is a good chance someone has already solved your problem.

To get the fastest help, help us help you.

### Support Request Template
When posting in **Support**, please include:

1.  **RV Details**: Year, Make, Model (e.g., *2022 Tiffin Allegro Bus 45OPP*)
2.  **Hardware**: Pi Model, CAN HAT type (e.g., *Pi 4, Waveshare RS485/CAN HAT*)
3.  **Software**: Home Assistant version, Libre Coach version.
4.  **The Problem**: "I expected X, but Y happened."
5.  **Logs**: Copy relevant lines from the Add-on logs.

:::tip[Logs]
Don't paste 500 lines of logs directly into the post. Use a code block (three backticks \`\`\`) or link to a Gist/Pastebin.
:::

## How to Share

### Sharing Dashboards
We love seeing what you claimed!
*   **Screenshots**: A picture is worth 1000 words.
*   **YAML**: Share your dashboard card configuration.
*   **Context**: Explain *why* you designed it that way.

### Sharing Automations
*   **Use Code Blocks**: Wrap your YAML in code blocks for readability.
*   **Dependencies**: Mention if your automation requires specific sensors (e.g., "Requires an Occupancy Sensor").

## Community Guidelines
*   **Be Nice**: We are all doing this for fun/free.
*   **Search First**: Your question might have already been answered.
*   **Give Back**: If someone helps you, verify the solution so others can find it.
